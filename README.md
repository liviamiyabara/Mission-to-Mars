# Mission-to-Mars
Module 10: Mission to Mars - Web Scraping with HTML/CSS

## Overview 

In this week, the student helped Robin and created a web scraping script to gather information about Mars; consolidating different forms of data (tables, news headlines, paragraphs and images) by using MongoDB and then show casing the information in a web application by using Flask. 

In the challenge, a script was added to scrape full-resolution images of Mar’s hemispheres and the titles of those images and updates were made to include the results in the web application:
